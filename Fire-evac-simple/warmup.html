Hi everyone! Are u all ready for the warm up? Let's begin!

We apologize for the poor graphics as Netlogo aims to be functional and not aesthetically appealing. Do follow through with this warm up so that you can enjoy your time fully on this page!

As you can see below, there are many controls with the model which will allow you to explore the model to a great extent. Please do not touch them yet! Instead, first let us introduce to you the overall interface of the model!

Key Features:

The map looks so colourful...but why is there a need to make it so colorful? Well, the colours represent different kinds of areas on the map! The colours represent:
Setup: Loads the simulation. Do be patient as the model may take some time to load!
Go: Starts the simulation! Let the evacuation begin!
Pause: Pauses the simulation.
Speed Slider: Adjusts the speed of the simulation. It is recommended that you slow it down!
Evac-time: The time taken for the last person to reach the brown patch.
Death-count: The number of people dead.
Total-evac-count: The number of people evacuated at a certain time
Time: The number of ticks at a certain point in the simulation. The conversion factor is 5 seconds for every tick. Number of students: Adjust the number of students in each simulation.

Hopefully the above description helps you navigate the model better! When you press setup, you realise that a very colourful map appears. The colourful map represents the floor plan and each area of the building is represented by a different colour. Do check out the features below to learn more!

Map Features:

The map looks so colourful...but why is there a need to make it so colorful? Well, the colours represent different kinds of areas on the map! The colours represent:
Light Blue: The inidvidual rooms that the residents stay in! Light Green: The corridor area in which residents can roam around in!
Light Purple: Restrooms!
Dark Purple: Student Lounges!
Orange: The exits of all rooms except the staircase exit.
Red: The staircase exit.
Dark Green: The stairwell.
White: Inacessible areas during a fire evacuation, including the lifts.
Black: The walls!
Brown: The area outside the building. Any minion that reaches a brown patch would be considered evacuated.

The colour representation is important in making sure that the minions move on the right path and be able to find their way to an exit. Also, it makes easier to track who has already evacuated out of the building.
